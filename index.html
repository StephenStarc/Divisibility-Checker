How it is calculated

Divisibility Check for 2

To check if a number is divisible by 2. You have get the last number and check if it is even. If it is Even that it is divisible and if it is Odd then it is not divisible

Divisibility Check for 3

For 3, Add all the number to together and Check if it is divisible by 3. Ex: if the number is 123 then 1 + 2 + 3 = 6. 6 is divisible by 3.

Divisibility Check for 4

Get Last two digit and check if it is divisible 4. Ex: 124 - the last two digits are 24 and 24 is divisible by 4.

Divisibility Check for 5

Check if the last digit ends with either 5 or 0. Ex: 125 - it ends with 5, so it is divisible by 5

Divisibility Check for 6

For 6, You just have to check it is divisible by both 2 and 3. Ex: 126 - Let's check if it is divisble by 2 - You have get the last digit which is 6, so it divisible by 2.
